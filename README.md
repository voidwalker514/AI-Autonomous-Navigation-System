# 🚀 AI-Based Autonomous Navigation System

## 📌 Overview

This project implements an AI-based autonomous navigation system using Python.
It simulates a robot navigating from a start point to a goal while avoiding obstacles using the **A* (A-Star) path planning algorithm**.

---

## 🎯 Key Features

* 🧭 Grid-based environment simulation
* 🚧 Random obstacle generation
* 🧠 A* path planning algorithm
* ⚠️ Obstacle avoidance
* 📊 Path visualization using Matplotlib

---

## 🛠️ Tech Stack

* Python
* NumPy
* Matplotlib

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
```

---

## 📸 Output

* The system computes the shortest path from start to goal
* Successfully avoids obstacles
* Displays a visual simulation of navigation
